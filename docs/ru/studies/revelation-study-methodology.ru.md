# Методология исследовательских статей по Откровению (RU)

Doc-Version: `1.0.0`  
Last-Updated: `2026-04-07`  
Source-Commit: `working-tree`

## Назначение

Этот документ превращает 7-этапную методологию из статьи `Предисловие` в рабочий стандарт для статьи по `Откр. 1:1-3` и для всех следующих статей по книге Откровение.

## Семь этапов

1. Формирование общего текста из всех первоисточников.
2. Словарь и структура предложений.
3. Выявление и анализ ветхозаветных аллюзий.
4. Откровение Иисуса Христа.
5. Анализ структуры и связей.
6. Исполнение пророчеств.
7. Живая вера в действии.

Практическая логика:

- этапы `1-2` отвечают за точное чтение текста;
- этапы `3-5` отвечают за смысл, контекст и глубину;
- этапы `6-7` отвечают за историю, применение и духовный вывод.

## Иерархия источников

Для этапов `3-7` приоритет источников должен быть постоянным:

| Приоритет | Источник | Роль |
| --- | --- | --- |
| `1` | Классические переводы | Межъязыковое сравнение текста |
| `2` | Jon Paulien | Главная методологическая и интерпретационная опора |
| `3` | Комментарии АСД | Конфессионально близкий комментарий |
| `4` | Официальные доктрины АСД | Догматические границы и официальный язык |
| `5` | Ellen G. White | Пророческое и духовное измерение |
| `6` | Другие авторы АСД и официальные уроки | Расширение поля и внутренняя проверка |
| `7` | Ранние реформаторы | История толкования и ранний протестантский слой |

Дополнительные правила:

- для этапа `1` первичны рукописи, транскрипции, коллации и текстологи;
- для этапа `2` первичны лексиконы, грамматики, морфология и синтаксис;
- для этапа `6` обязательна внешняя историческая проверка через церковных и светских историков.

## Правила доказательной работы

- Каждая прямая цитата в будущей статье должна сопровождаться конкретной ссылкой.
- Для Jon Paulien, Ellen White и `SDA Bible Commentary` по возможности указывать главу, страницу, параграф или стиховой блок.
- Каждый содержательный абзац статьи должен иметь собственную карту доказательств: `тезис`, `источник`, `тип вывода`, `уровень уверенности`.
- Нельзя строить богословский вывод раньше, чем завершены этапы `1-2`.
- Нельзя строить историческое исполнение раньше, чем завершены этапы `3-5`.
- Для спорных исторических утверждений этапа `6` нужен двойной контроль:
  - один первичный или близкий к первичному исторический источник;
  - один академический контрольный источник.
- Слабые зеркала и сомнительные PDF-хранилища не входят в обязательное ядро.

## Шаблон сборки каждой статьи

Для каждого отрывка нужно собрать один и тот же пакет:

1. `Text Dossier`
2. `Language Dossier`
3. `OT Matrix`
4. `Christ Matrix`
5. `Structure Map`
6. `History Matrix`
7. `Faith/Application Sheet`

Затем статья строится слоями:

1. краткое введение;
2. основной блок этапов `1-5`;
3. историко-пророческий блок этапа `6`;
4. пасторско-духовный блок этапа `7`;
5. визуальные, аудио- и видео-вставки.

## Первый рабочий проход именно для Откр. 1:1-3

Следующий порядок работы:

1. Расширить уже готовый этап `1` через `CNTR`, `NTVMR`, `CSNTM`, `Codex Sinaiticus`, `ECM`.
2. Построить этап `2` вокруг слов `αποκαλυψις`, `εδωκεν`, `εσημανεν`, `μαρτυρια`, `τηρουντες`, `καιρος`, `εγγυς`.
3. На этапе `3` в первую очередь проверить фон `Дан. 2`, `Дан. 12` и формулы заветного послушания.
4. На этапе `4` показать цепочку откровения: `Бог -> Христос -> ангел -> Иоанн -> церковь`.
5. На этапе `5` показать, как `Откр. 1:1-3` программирует весь пролог книги.
6. На этапе `6` отдельно собрать историю толкования слов `вскоре` и `время близко`.
7. На этапе `7` вернуть текст к формуле `читать -> слушать -> соблюдать`.

Минимальный визуальный пакет:

- фрагменты рукописей `01`, `02`, `04`, `046`, `025`;
- таблица разночтений;
- схема цепочки откровения;
- таблица ключевых греческих слов;
- карта OT-аллюзий;
- схема структуры пролога;
- историческая шкала интерпретаций `вскоре / близко`.

## Финальная установка

- Сначала текст.
- Потом язык.
- Потом ветхозаветный фон.
- Потом Христос.
- Потом структура.
- Потом история исполнения.
- Потом живая вера.

И на каждом этапе Jon Paulien должен оставаться главным спутником исследования.
